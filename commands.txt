1. git init — Initialize a new Git repository.
2. git clone — Copy a remote repository to your machine.
3. git status — Show the working tree status.
4. git add — Stage changes for commit.
5. git commit — Record staged changes to the repository.
6. git push — Upload local commits to a remote repository.
7. git pull — Fetch and merge changes from a remote repo.
8. git branch — List, create, or delete branches.
9. git checkout — Switch branches or restore files.
10. git merge — Merge another branch into the current branch.
